## HR/招聘知识 - 试用期辞退风险

### R-002《面试人员考核表》核心要点

**问题：** 试用期辞退员工难？容易赔 N+1？

**解决方案：** 面试时填写《面试人员考核表》要注意三个细节：

1. **录用标准固化**
   - 在"试用期考核重点"一栏明确写出 KPI 或技能要求
   - 员工转正前没做到可合法辞退

2. **关键事件记录**
   - 记录面试者吹过的牛、承诺的资源、展示的作品
   - 入职后发现货不对板，作为"提供虚假信息"证据

3. **薪资与背调联动**
   - 录用建议与建议薪资定级关联
   - 明确勾选是否需要背调

### 法律依据
- 试用期辞退必须证明"不符合录用条件"
- 《面试人员考核表》是"录用条件"的原始凭证

## 技术知识 - SpringAI 智能知识库

### 基于 SpringAI 的企业级智能知识库问答系统

**来源：** 微信公众号 - 开源工厂  
**链接：** https://mp.weixin.qq.com/s/pfiKwyCEuobgJyQnwLK9-g

#### 01 项目介绍
本项目是一款基于 Spring Boot+Spring AI+Ollama 构建的智能知识库问答系统，核心实现了检索增强生成（RAG）功能，是开发者学习 AI 落地、企业快速搭建专属知识库的优质项目。

#### 02 核心优势

1. **Java 生态无缝衔接**
   - 基于 Spring Boot、Spring AI 开发
   - 无需跨栈学习，适配企业现有 Java 技术栈

2. **本地化部署，数据隐私全可控**
   - 集成 Ollama 本地大语言模型服务
   - 无需依赖云端 API，避免数据隐私泄露

3. **模块化设计，扩展灵活易定制**
   - 解耦式架构
   - 支持新增文档格式、集成更多 AI 模型

4. **开箱即用，轻量化部署**
   - 标准 Maven 项目构建流程
   - 支持响应式 Web 界面

#### 03 应用场景

- 企业内部知识管理
- 技术团队研发支持
- 智能客服辅助
- 教育培训知识答疑
- 金融/法务专业咨询

#### 04 技术栈

**后端：**
- Spring Boot 3.4.2
- Spring AI 1.0.0
- Spring Security + JWT
- MyBatis-Plus 3.5.11
- ElasticSearch 8.x
- WebSocket
- Spring AI Tika、FlexMark

**前端：**
- HTML/CSS/JavaScript
- Marked.js
- 响应式设计
