# SpringAI 企业级智能知识库问答系统\n\n## 项目介绍\n基于 Spring Boot+Spring AI+Ollama 构建的智能知识库问答系统，核心实现检索增强生成（RAG）功能。\n\n## 核心优势\n\n### 1. Java 生态无缝衔接\n- 基于 Spring Boot、Spring AI 开发\n- 适配企业现有 Java 技术栈\n\n### 2. 本地化部署，数据隐私全可控\n- 集成 Ollama 本地大语言模型服务\n- 避免数据隐私泄露\n\n### 3. 模块化设计，扩展灵活\n- 解耦式架构\n- 支持新增文档格式、集成更多 AI 模型\n\n### 4. 开箱即用，轻量化部署\n- 标准 Maven 项目构建流程\n- 支持响应式 Web 界面\n\n## 应用场景\n- 企业内部知识管理\n- 技术团队研发支持\n- 智能客服辅助\n- 教育培训知识答疑\n- 金融/法务专业咨询\n\n## 技术栈\n\n**后端：**\n- Spring Boot 3.4.2\n- Spring AI 1.0.0\n- Spring Security + JWT\n- MyBatis-Plus 3.5.11\n- ElasticSearch 8.x\n- WebSocket\n\n**前端：**\n- HTML/CSS/JavaScript\n- Marked.js\n- 响应式设计\n\n## 环境要求\n- Java 17+\n- MySQL 5.7+\n- Maven 3.6+\n- ElasticSearch 8.x\n- Ollama 服务\n\n## 产品功能\n1. 智能对话：自然交互，实时响应\n2. 知识库管理：多库创建，权限管控\n3. 文档管理：多格式支持，自动向量化\n4. 对话管理：会话留存，历史可查\n5. RAG 检索增强：精准匹配，智能生成\n6. 系统管理：配置灵活，用户隔离\n\n## 安装步骤\n1. 克隆项目\n2. 数据库初始化\n3. 环境配置\n4. 构建并运行\n5. 访问系统 (http://localhost:8080)\n- 默认账户：admin/admin\n\n来源：微信公众号 - 开源工厂
