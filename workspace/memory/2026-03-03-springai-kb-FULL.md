01 项目介绍：SpringAI 生态的 RAG 实战利器 还在为企业知识零散、检索效率低烦恼？还在纠结如何快速搭建本地化 RAG 知识库系统？今天给大家推荐一款基于 SpringAI 开发的开源智能知识库问答系统 本项目是一款基于 Spring Boot+Spring AI+Ollama 构建的智能知识库问答系统，核心实现了 检索增强生成（RAG） 功能，是开发者学习 AI 落地、企业快速搭建专属知识库的优质项目。作为一款非 demo 级的学习与实战项目，它并非简单的功能堆砌，所有需求和功能均保证生产级基本可用，同时紧跟 SpringAI 1.0.0 稳定版的技术迭代，持续更新依赖版本，兼顾技术前沿性和项目实用性。无论是个人开发者学习 RAG 技术落地，还是中小企业快速搭建私有化智能知识库，都是绝佳选择。

02 核心优势：四大亮点直击开发与使用痛点

*   Java 生态无缝衔接，开发零门槛

基于 Spring Boot、Spring AI 开发，完全贴合 Java 后端开发者的技术习惯，无需跨栈学习，熟悉 Spring 生态即可快速二次开发，适配企业现有 Java 技术栈。

*   本地化部署，数据隐私全可控

集成 Ollama 本地大语言模型服务，无需依赖云端 API，彻底规避数据隐私泄露风险，金融、医疗、政务等隐私敏感场景也能放心使用，同时降低云端调用的成本和延迟。

*   模块化设计，扩展灵活易定制

系统采用解耦式架构，从模型配置、检索参数到功能模块，均可灵活调整和扩展，支持新增文档格式、集成更多 AI 模型、自定义检索算法，满足不同企业的个性化需求。

*   开箱即用，轻量化部署

无需复杂的环境配置，遵循标准的 Maven 项目构建流程，配套详细的部署文档，新手也能快速完成本地搭建，同时支持响应式 Web 界面，电脑、移动端均可便捷操作。

03 应用场景：多领域落地，释放知识价值 企业内部知识管理 将企业制度、产品手册、技术文档、流程规范等统一接入知识库，员工通过自然语言提问即可快速检索答案，告别翻找文档堆、微信群答疑的低效模式，提升办公效率。技术团队研发支持 研发团队可搭建专属技术知识库，整合技术文档、开发规范、问题解决方案等，新人快速上手，老员工高效解决开发问题，实现技术知识的沉淀与复用。智能客服辅助 对接企业客服场景，将常见问题、产品咨询、售后流程等录入知识库，为客服人员提供实时问答支持，提升客服响应效率和回答准确性，降低人工重复工作。教育培训知识答疑 教育机构可搭建学科知识库、培训资料库，学员通过自然语言提问获取知识点解析、习题解答，实现个性化学习；企业内部培训也可借助其完成员工知识考核与答疑。金融 / 法务专业咨询 金融、法务等专业领域可搭建专属知识库，整合政策法规、行业规范、案例资料等，支持专业问题的智能检索与解答，辅助从业人员快速做出决策。

04 技术栈：前后端全栈适配，技术选型主流 后端核心技术栈

*   应用框架：Spring Boot 3.4.2

*   AI 集成框架：Spring AI 1.0.0（Java 生态 AI 开发核心框架）

*   安全认证：Spring Security + JWT

*   数据持久化：MyBatis-Plus 3.5.11

*   向量存储 / 检索：ElasticSearch 8.x（支撑 RAG 核心检索能力）

*   实时通信：WebSocket（实现 AI 回答流式响应）

*   文档解析：Spring AI Tika、FlexMark（Markdown 处理）

前端技术栈

*   基础技术：HTML/CSS/JavaScript

*   Markdown 解析：Marked.js

*   界面设计：响应式设计，适配电脑、移动端等多种设备

AI 服务支持

*   本地大模型：Ollama（支持 deepseek-r1:32b、bge-m3 等模型）

*   通用适配：OpenAI 兼容 API，支持多种 AI 服务提供商

环境要求 Java 17+、MySQL 5.7+、Maven 3.6+、ElasticSearch 8.x、Ollama 服务（或其他兼容 AI 服务）

05 产品功能：六大核心能力，打造完整知识管理闭环 智能对话：自然交互，实时响应

*   基于 Ollama 实现自然语言交互，支持流式响应，AI 回答实时显示，提升体验；

*   多模型自由切换，可配置默认对话模型和嵌入模型，适配不同问答场景；

*   支持选择指定知识库问答或普通通用对话，灵活切换问答模式。

知识库管理：多库创建，权限管控

*   支持创建、查看、编辑、删除多个知识库，实现知识分类管理；

*   基于角色的访问控制，实现知识库权限管理，保障知识安全；

*   每个知识库独立管理，支持查看知识库基本信息和文档统计。

文档管理：多格式支持，自动向量化

*   支持 PDF、Word、TXT、Markdown 等多种格式文档上传，支持批量处理；

*   文档上传后自动解析内容并完成向量化存储，为 RAG 检索提供基础；

*   支持文档预览、编辑、删除，Markdown 格式原生渲染，查看更友好。

对话管理：会话留存，历史可查

*   可创建多个对话会话，支持对话标题自定义修改，方便分类管理；

*   保存对话历史记录，支持历史会话搜索，快速找回过往问答内容；

*   对话过程中自动保持上下文，实现连续自然的交互体验。

RAG 检索增强：精准匹配，智能生成

*   基于 ElasticSearch 实现高效向量存储与检索，支持余弦相似度匹配；

*   智能文档检索与相关性匹配，可调节相似度阈值，优化检索效果；

*   检索结果与 AI 生成内容有机结合，解决纯大模型 “知识幻觉” 问题，回答更精准。

系统管理：配置灵活，用户隔离

*   完善的 JWT 身份认证与授权体系，支持用户注册、登录、管理；

*   实现用户数据隔离，不同用户的知识库、对话记录相互独立；

*   支持平台设置，可配置默认 AI 模型、管理多种模型，调整系统核心参数。

06 界面实拍：简洁直观，操作友好 智能对话界面 支持自然语言输入，可选择知识库、开启深度思考 / 联网搜索，AI 回答实时流式显示，对话历史右侧可快速查看，操作栏支持新建对话、管理会话，极简交互。知识库管理界面 可快速查看已创建的知识库列表，支持新建、编辑、删除知识库，每个知识库显示核心信息和文档数量，一键进入文档管理，知识分类一目了然。

文档管理界面 支持添加文档、查看文档列表，每个文档显示格式、创建时间，可一键预览、编辑、删除，批量处理功能提升文档管理效率。

模型设置界面 可配置默认对话模型和嵌入模型，支持添加、编辑、删除多种 AI 模型，模型信息（名称、路径、类型）清晰展示，一键启用 / 禁用，模型切换灵活。

07 安装教程：五步快速搭建，新手也能上手 项目遵循标准的 Maven 项目部署流程，配套详细的配置文档，只需五步即可完成本地搭建，全程无复杂操作：

1.   第一步：克隆项目

打开终端，执行命令克隆项目到本地，进入项目目录：2. 第二步：数据库初始化 在 MySQL 中创建指定数据库，数据表将在应用首次启动时自动创建，若未自动创建可使用项目内 sql/schema.sql 文件手动创建：

CREATE DATABASE ai_knowledge_base CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;3. 第三步：环境配置 编辑项目中 src/main/resources/application-dev.yml 文件，替换以下关键配置的占位符为实际信息（详细说明参考项目 DEPLOY.md）：

*   ElasticSearch 配置：服务地址、用户名、密码；

*   数据库配置：MySQL 连接 URL、用户名、密码；

*   文件上传目录：指定实际的文件存储绝对路径；

*   JWT 配置：设置长随机字符串作为秘钥，保障认证安全。

4. 第四步：构建并运行项目 执行 Maven 命令构建项目，随后启动应用，等待项目启动成功：

# 构建项目 mvn clean install# 启动应用 mvn spring-boot:run 5. 第五步：访问系统 打开浏览器，输入地址 http://localhost:8080 ，使用默认管理员账户登录即可开始使用：

*   用户名：admin

*   密码：admin（也可通过注册页面创建新用户账户）
